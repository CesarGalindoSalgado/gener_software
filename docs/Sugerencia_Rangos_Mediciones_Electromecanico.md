# Sugerencia de rangos de referencia para mediciones — Rutinas electromecánicas (G-ener)

> **Propósito.** Como buena práctica de control de calidad, G-ener sugiere definir un **rango aceptable** para los pasos de medición de cada rutina. Mientras el cliente no defina estos rangos, el técnico determina en sitio si la lectura **cumple / no cumple**. Este documento cubre el alcance **electromecánico** y lista los pasos candidatos por rutina, con la unidad sugerida y un espacio para el rango. Es una recomendación profesional, no un requisito.

Los pasos marcados **[YA DEFINIDO]** ya traen rango en el documento fuente.
